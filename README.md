## Prerequisites

- Node.js
- MongoDB
- Git


```sh
git clone <repository-url>
```
## Backend Setup
```sh
cd backend
npm install
```

### Create a .env file in the backend directory:

```sh
PORT=5000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
```
### For JWT srcret
```sh
npm run jwt
```

## Frontend Setup
